## 🚀 Deploy in 3 Steps

### Step 1: Push to GitHub (5 min)
```bash
git init
git add .
git commit -m "SweetOven ready for deployment"
git push origin main
```

### Step 2: Deploy Backend on Render (5 min)
- https://render.com → New Web Service
- Select repo → backend directory
- Add environment variables (see QUICK_DEPLOY.md)
- Deploy!

### Step 3: Deploy Frontend on Vercel (5 min)
- https://vercel.com → New Project
- Select repo → frontend directory
- Add REACT_APP_API_URL variable
- Deploy!

**Total: 15 minutes** ✅

## ⚡ Performance Tips

- **First backend request slow?** (Render free tier wakes up)
  - Solution: Keep using your app, or upgrade to paid tier

- **Image upload slow?** (Cloudinary optimizes)
  - Solution: Normal, subsequent loads are cached

- **Database slow?** (Free tier 512MB)
  - Solution: Upgrade if you go over limit
